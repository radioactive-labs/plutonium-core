# Installation

## Requirements

Plutonium requires the following to run:

- Ruby 3.2.2+
- Rails 7.1+

## Installing Plutonium

1. Add plutonium to your bundle

```bash
bundle add plutonium
```

2. Run the install generator

```bash
rails g pu:core:install
```
